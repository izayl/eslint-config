# @izayl/eslint-config

shareable eslint config preset for my personal project

## Install

```sh
yarn add -D @izayl/eslint-config
```

## Usage

Just add it to `package.json`:

```json
{
  "name": "my-awesome-package-name",
  "eslintConfig": {
    "extends": "@izayl/eslint-config"
  }
}
```

or use `.eslintrc`:

```json
{
  "extends": "@izayl/eslint-config"
}
```
